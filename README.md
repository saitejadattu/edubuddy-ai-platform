# EduBuddy AI - Learning Platform

## Overview
EduBuddy AI is an intelligent learning companion platform designed for academic excellence. It provides comprehensive study tools, exam preparation resources, and progress tracking features for students of all levels.

## Features

### Core Modules
- **5-10 Class**: Foundation & Board Exams preparation
- **11-12 Class**: JEE, NEET, Boards & More
- **BTech**: Engineering & Tech Courses
- **State Govt Exams**: Telangana & AP Exams
- **Central Govt Exams**: UPSC, SSC, Railways & More

### Study Tools
- **Guidelines**: Comprehensive study materials
- **Time Management**: Game Schedule Planner
- **Organize Notes**: AI Summaries from Videos/Files
- **Personal Dashboard**: Track Progress & Analysis
- **Create Test**: Custom test creation
- **Previous Papers**: Access past exam papers
- **Mistake Analysis**: Learn from errors
- **Expected Questions**: Predict likely exam questions
- **Exam Calendar**: Manage exam schedules

### Additional Features
- **Chat Assistant**: AI-powered learning companion
- **Transport Facility**: Bus Timing & Route Planner
- **Financial Guidance**: Budget & Fee Planning
- **Health & Diet Planner**: Personalized wellness plans

## Tech Stack
- **Frontend**: React with Base44 UI Framework
- **Backend**: Base44 Integration
- **Authentication**: Base44 Auth Service
- **Email**: Core SendEmail Integration
- **Database**: Base44 Entities

## Installation

1. Clone the repository:
```bash
git clone https://github.com/saitejadattu/edubuddy-ai-platform.git
cd edubuddy-ai-platform
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```
